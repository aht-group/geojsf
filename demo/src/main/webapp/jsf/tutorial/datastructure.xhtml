GeoJSF Data Structure

The datastructure is implemented with generic interfaces and a default implementation. It's possible to use a completly custom implmentation, the default implementation or a mixture of them. Also Entity Java Beans can easily implement these generic interfaces and a custom management web page for all three entities is possible. With this page the user is allowed to register layers, or create and modify custom views.
