GeoServer and JBoss AS7

One download format of GeoServer is a Web Archive. This should run without problems on Tomcat. While deploying geoserver.war on JBoss AS the java.lang.ClassNotFoundException: com.sun.imageio.spi.FileImageInputStreamSpi occurs.

Modified geoserver.war
